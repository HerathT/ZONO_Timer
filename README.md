# ZONO Timer Web App 🖥️⏱️

A **responsive, interactive Timer Web App** developed by **Kanchana Herath**, designed to manage time effectively for educational or general purposes. Ideal for **physical examinations**, classrooms, or personal productivity, it allows users to track current time, measure elapsed time, or countdown with precision.  

---

## Website Overview

This web app operates in **three main modes**:

### 1. Current Time Clock 🕒
- Displays the **real-time local time** or a **selected time zone**.  
- Helps track time differences during exams or online events.

### 2. Stopwatch ⏱️
- Counts **upwards from 00:00:00** to measure elapsed time.  
- Useful for timing activities, presentations, or individual tasks.

### 3. Timer ⏰
- Counts **down from a set time**.  
- Users can **increase/decrease seconds, minutes, or hours** before starting.  
- Ensures accurate timing for exams or tasks.

### Additional Features
- **Theme Toggle (Dark/Light) 🌙☀️** – switch for better visibility in different lighting conditions.  
- **Color Picker 🎨** – customize interface colors for classroom or projector visibility.  
- **Navigation Menu with Modals 📑** – includes **About**, **Portfolio**, **Other Sites**, and **Games**.  
- **Responsive Design 📱💻** – works on desktops, laptops, tablets, and mobile screens.  
- **Time Zone Selection 🌐** – select any time zone to display current time accurately.

---

## Technologies Used 💻
- **Languages:** HTML5, CSS3, JavaScript (ES6+)  
- **Libraries & Frameworks:** Font Awesome (for icons)  
- **Tools & Features:**  
  - Modals for additional content  
  - Interactive buttons (start, pause, reset, adjust time)  
  - Color picker and theme toggle  
  - Responsive layout for multiple devices  

---

## Pros ✅
- **Multi-functional** – Clock, Stopwatch, and Timer in one platform.  
- **Highly Customizable** – Theme, colors, and time adjustments.  
- **Projector-Friendly** – Large, clear, scalable fonts for classrooms or exams.  
- **Interactive UI** – Buttons, overlays, and modals enhance usability.  
- **Easy Controls** – Start, pause, reset, and adjust time effortlessly.  
- **Supports Exams 🎓** – Shows remaining or current time clearly for participants.  

---

## Cons ⚠️
- **Requires Internet** for some features (icons, fonts).  
- **Single Page Layout** – Advanced scheduling or multiple timers not supported.  
- **Manual Timer Adjustment** – Requires clicking buttons rather than typing time.  
- **No User Authentication** – Cannot save sessions or track multiple users.  

---

## Advantages for Examinations 🎓
- **Centralized Timing** – Teacher can project the timer for all students.  
- **Live Clock Display** – Keeps participants aware of current time.  
- **Countdown Timer** – Ensures fairness; everyone starts and finishes together.  
- **Stopwatch Mode** – Measure activity durations if needed.  
- **Customizable Colors & Themes** – Makes visibility clear on projector screens.  
- **Simple Interface** – No complex menus; easy to operate in exam conditions.  

---

## Summary
The **Timer Web App** is perfect for **educational settings**, especially **physical exams using projectors and computers**. It provides **clear visibility**, **interactive controls**, and **accurate time tracking**, making it a reliable tool for teachers and students alike. 🎯🕰️  

---

## How to Use
1. Open the website in a **modern browser**.  
2. Select the **mode**: Clock 🕒, Stopwatch ⏱️, or Timer ⏰.  
3. Use the **controls** (start, pause, reset) as needed.  
4. Adjust time or colors using the **adjustment buttons** and **color picker**.  
5. Toggle between **light/dark themes** for better visibility.
